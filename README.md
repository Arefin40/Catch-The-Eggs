# Catch The Eggs Game

## Gameplay

### Game Points

| Entity     |  Points |
| :--------- | ------: |
| Golden Egg |   `+10` |
| Blue Egg   |    `+5` |
| Normal Egg |    `+1` |
| Poop       |   `-10` |
| Black Bomb | `Death` |

### Game Perks

- **Basket Enlargement** → increases basket width for 10 seconds.
- **Slow Fall** → reduces egg falling speed for 5 seconds.
- **Time Extension** → adds 5 seconds extra game time.

### Game Controls

- Left Arrow `←` / `A` → Move basket left.
- Right Arrow `→` / `D` → Move basket right.
- Move mouse horizontally → to control basket position.

### Game Over

- **Game Time**: 2 minutes (120 seconds).
- Game ends when timer runs out or a bomb is caught.

### Keyboard Controls

- Press `enter` to start the game.
- Press `r` to restart the game.
- Press `h` for help menu.
- Press `p` to play/pause the game.
- Press `esc` to return to the main menu.
